HOW TO UPLOAD ON HOSTINGER
---------------------------

1. Log in to Hostinger hPanel
2. Go to: Hosting → Manage → File Manager
3. Navigate to: public_html/
4. IMPORTANT: Enable "Show hidden files" (to see .htaccess)
5. Delete any existing WordPress files in public_html/
6. Upload ALL files EXCEPT README.txt
7. Verify the site loads at https://trishikacarrentalhubli.in/


AFTER UPLOAD — CHECKLIST
--------------------------

□ Site loads correctly on desktop and mobile
□ HTTPS works (padlock shows in browser)
□ Phone links work (tap to call on mobile)
□ WhatsApp button opens correctly
□ Booking form submits to WhatsApp
□ Rate estimator calculates correctly
□ Google Map loads in the Find Us section
□ robots.txt accessible at: trishikacarrentalhubli.in/robots.txt
□ sitemap.xml accessible at: trishikacarrentalhubli.in/sitemap.xml


GOOGLE SEARCH CONSOLE — AFTER UPLOAD
--------------------------------------

1. Verify ownership of trishikacarrentalhubli.in in GSC
   (Use DNS TXT method via Hostinger hPanel → DNS Zone)

2. Submit sitemap:
   GSC → Sitemaps → https://trishikacarrentalhubli.in/sitemap.xml

3. Request indexing of homepage:
   GSC → URL Inspection → paste URL → Request Indexing


SNEHACABS MIGRATION — AFTER TRISHIKA IS LIVE
----------------------------------------------

Only do this AFTER confirming Trishika site is fully live and working.

Step 1: Go to Snehacabs hosting on Hostinger
Step 2: Open public_html/.htaccess
Step 3: Replace entire content with:

  Options +FollowSymLinks
  RewriteEngine On
  RewriteRule ^(.*)$ https://trishikacarrentalhubli.in/ [R=301,L]

Step 4: In Snehacabs Google Search Console:
  Settings → Change of Address → Select Trishika property → Submit

Step 5: Update Google Business Profile website URL to:
  https://trishikacarrentalhubli.in/
